Abstract

Controlled vocabulary for the classification of the theme for a mobility dataset.

Data content categories and sub-categories

The taxonomy is organised in two hierarchy levels. The first level represents a set of "Data content category" describing the classification of the data content on an aggregated level. The second level “Data content sub category”, concretises the "Data content category" via one or several sub-categories.
